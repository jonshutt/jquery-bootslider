Gallery 1

This gallery shows just one item at at time. The number of items in view are purely controled by standard bootsrap column classes

$( "#gal1" ).bootslider();

Gallery 2

This is a standard click and drag gallery, sliding to the nearest item when released

$( "#gal1" ).bootslider();

Gallery 3

This is a standard click and drag gallery, and stops where you leave it

$( "#gal3" ).bootslider({snapToItem:  true});

Gallery 4

This gallery adds next and previous buttons

$( "#gal4" ).bootslider({buttons:true});

Gallery 5

This gallery links items on the page to create custom next and previous button.

$( "#gal5" ).bootslider({next:  '#next', previous: '#previous'});

Gallery 6

This gallery adds a callback, so events can be trigged after gallery has slide to a new position. In this case it reads a data attribute on the slide

$( "#gal6" ).bootslider({
	complete:function(e) {			
		$('#response').html(e.firstChild.data('details') + ' is the first item in view');				
	}	
});
